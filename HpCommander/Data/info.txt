HP Commander — House Party console command builder

This tool is a work in progress and almost certainly has rough edges. A lot of
the command syntax was pieced together from the game's own files and help text,
but the console is fiddly and some commands may not behave exactly as expected.

If something is wrong or missing, please report it — bug reports and corrections
are genuinely helpful and shape what gets fixed next. Use the "Report an issue"
button below (or the Issues tab on the GitHub repo).
